Workers Compensation Calculator

== Description ==

Calculator that allows user to select different body parts, their rating and receive an estimate for their workers compensation payment.

== Installation ==

Installing "Palace Law Calc" can be done either by searching for "Palace Law Calc" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress

== Updating Injury Data

Injury data is stored in `includes/injury_data.tsv` and the database is refreshed from it whenever the plugin is activated.


Edit the spreadsheet here and then export as `injury_data.tsv`
https://docs.google.com/spreadsheets/d/1Z6AYfNLiETNTjRgwe02iw-AF3fo62_w-q4uL90v4zFc/edit?usp=sharing
